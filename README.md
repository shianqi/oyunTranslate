# oyunTranslate（奥云字典）

## 所用技术

* [npm](https://www.npmjs.com/) （选用）
* [nodejs](https://nodejs.org) （选用）
* [browser-sync](https://browsersync.io/) （选用）
* [AngularJS](http://angularjs.org) (主要开发框架) (v1.4.8)
* [purecss](http://www.purecss.org/) （样式框架）

## 运行项目

在 cmd 中 cd 到该目录，运行

```bash
npm run start
```

## 项目目录说明

* css
  * layouts  字体和样式表
* font  字体
* img  图片
* js  脚本
* view1  首页界面
* view2  汉文→传统蒙古文
* view3  传统蒙古文→汉文
* view4  汉文→西里尔蒙古文
* view5  西里尔蒙古文→汉文

**详细情况** 请查看代码注释
